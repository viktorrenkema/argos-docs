# Invoiced customers

Discover how invoiced GitHub accounts affect subscribing to Argos' paid plan and explore alternative subscription options.

If you have a GitHub invoiced account and are looking to subscribe to Argos' paid plan through the GitHub Marketplace, you may have encountered the following error message:

> "Unfortunately, invoiced customers cannot purchase paid plans on the GitHub Marketplace."
>
> **— GitHub error message**

This message means that if your GitHub account is set up to receive an invoice for your subscription (instead of charging your credit card directly), you cannot use the GitHub Marketplace to subscribe to Argos.

{% hint style="info" %}
Not to worry—you can still purchase an Argos paid plan by subscribing via Stripe: https://argos-ci.com/docs/how-to-subscribe
{% endhint %}
